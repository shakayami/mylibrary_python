## numfunc
- エラトステネスのふるいO(NloglogN)
- メビウス関数O(NloglogN)
- 素因数分解O(√N)
- 約数列挙O(√N)

## modint
- 二項係数mod 素数　前計算O(N),クエリO(1)
- mod素数のときの逆元O(log mod)
## bostan_mori_algorithm
- [x^N] P(x)/Q(x)をO(polyprod(P,Q)logN)で計算する
